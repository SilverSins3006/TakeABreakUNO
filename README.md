# TakeABreakUNO

A web - application to encourage taking a break.
Tristan Kuhnel
Danie Cunningham
Micheal
Cassia Nebel
A web application that begins tracking activity as soon as the user opens the program. Once the program starts, several menus appear, allowing the user to choose how often challenge intervals occur, the difficulty of the challenges, and the types of challenges they would like to receive. These challenges can range from finding a spoon, taking a shower, solving a puzzle, going somewhere, or simply watching something. The goal is to promote positive behaviors such as going outside, completing chores, and taking better care of yourself away from the computer.

The application also includes a timer that tracks how long the user has been online since opening the program, as well as a counter that records how many challenges have been completed. This provides users with a visual representation of their screen time and encourages them to take regular breaks throughout the day.

React / node.js
Using JavaScript

## Getting started

### 1. Install Node.js

If you do not already have Node.js installed, install a recent version from:

- https://nodejs.org/

### 2. Clone the repo and install dependencies

```bash
git clone https://github.com/SilverSins3006/TakeABreakUNO.git
cd TakeABreakUNO/client
npm install
```

### 3. Start the development server

```bash
npm run dev
```

Then open the local URL shown in the terminal, usually `http://localhost:5173`.

## Common commands

- `npm run dev` — start the app locally (most code changes reload automatically)
- `npm run build` — build the app for production
- `npm run preview` — preview the production build locally
